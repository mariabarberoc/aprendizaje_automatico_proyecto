Loudness

Representa la sonoridad global de una cancin medida en decibelios (dB). Este valor se calcula como una media a lo largo de toda la pista y permite comparar la intensidad sonora entre diferentes canciones.
Los valores de Loudness suelen estar comprendidos entre 60 dB y 0 dB:
Canciones con valores alrededor de 60 dB son muy suaves o acsticas.
Canciones con valores cercanos a 0 dB son muy intensas, comprimidas y tpicas de gneros ms comerciales o electrnicos.

def classify_tempo(bpm):
    if bpm < 24:
        return 'Larghissimo'
    elif bpm < 46:
        return 'Grave'
    elif bpm < 61:
        return 'Lento / Largo'
    elif bpm < 77:
        return 'Adagio'
    elif bpm < 109:
        return 'Andante'
    elif bpm < 121:
        return 'Moderato / Allegretto'
    elif bpm < 157:
        return 'Allegro'
    elif bpm < 177:
        return 'Vivace'
    elif bpm < 201:
        return 'Presto'
    else:
        return 'Prestissimo'

Durante el proceso de limpieza, eliminamos las variables 'views', 'likes' y 'comments', estas columnas representan medidas de interaccin y popularidad obtenidas posteriormente a la publicaicon del contenido, es por esto que no resultan adecuadas para el objetivo de nuestro proyecto.

El propsito del anlisis es predecir eel nmero de reproducciones en Sportify(stram) utilizando nicamente informacin disponible antes del lanzamiento, fundamentalmente caractersticas musicales y metadatos intrnsecos de la cancin.

En caso de usar estas metricas podriamos tener problemas de fuga de informacin (Data Leakage), estas variables no estn disponibles antes de su publicacin y reflejan su rendimiento real o falta de relacin causal con las caractersticas musicales, no aportan informacin sobre los atributos sonoros que buscamos analizar, y mezclan xito comercial con interaccin social.

Por estas razones, estas variables se han eliminado para asegurar un modelo ms robusto, coherente y predictivamente vlido, preservando nicamente informacin relevante para analizar la msica y su potencial de xito.
